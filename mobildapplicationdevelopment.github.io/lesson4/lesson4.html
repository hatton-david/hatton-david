AJAX

AJAX stands for Asynchronous JavaScript and XML. AJAXJAX uses one object that comes with JavaScript. The object lets you load an external file and add its content to your webpage.

One way we can use AJAX in an XMLHttpRequest. You can do many things with an XMLHttpRequest. Fir example.
  • Update a web page without reloading the page
  • Request data from a server after the page has loaded
  • Receive data from a server after the page has loaded
  • Send data to a server in the background

For example, you can get the date when the page was last modified



With AJAX you can also access external files that have been uploaded to the internet, from your webpage.
