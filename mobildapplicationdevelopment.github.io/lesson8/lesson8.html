CSS Transitions

CSS transitions let you change property values over a difined duration and in a smooth manner. Javascript can be used to facilitate that transition.

you need two basic things to create a transition effect. Firstly, you need a CSS property to add the effect to. then you will need to decide the duration of the effect. In the example below, I use javascript to trigger CSS transitions that effect the height of the box, as well as the color of the box. I can also use JavaScript functions to control the speed of the transition.




CSS Animations

CSS animations use something called @keyframes to tell each element will have which styles and at what time. We can also add loops so that the animation will continue forever, or as long as we would like. Javascript can be used to trigger these animations as well.
