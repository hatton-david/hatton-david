HTML events

An HTML events anything that happens to or anything HTML elements can do. When JavaScript is used with HTML it can trigger some of these events.

In my example below, I used JavaScript to changes the content of the "p" element. When you click the button, the contents will be replaced with the current date.

Click the button to display the date.

In this example, the img element changes size when you hover your mouse over it.

Here is a list of some of the most common HTML events.

  • onchange: An element has been changed
  • onclick: Someone clicks theelement
  • onmouseover: Someone hovers the mouse over an HTML element
  • onmouseout: Someone moves the mouse away from the element
  • onkeydown: Someone pushes a keyboard key
  • onload: The browser has finished loading the page

Touch events are events that happen when someone touches something like a mobile device or touch screen.

There are 4 basic touch events

  • touchstart - a touch event has started
  • touchend - a touch event has ended
  • touchmove - the finger moves over the surface
  • touchcancel - the touch event has been cancelled.
