When storing and retrieving simple data, HTML web storage has two different objects localStorge and sessionStorage.

localStorage Object

The localStorage object stores data that will not expire, meaning The data will not deleted when whenm you close the browser. It is done by createing a localStorage name/value pair like with name="lastname" and value="Hatton". You would store the information like this.

localStorage.setItem("lastname", "Hatton");


And retrieve it by calling the first paramiter. In this case it is lastname. It would look like this.



localStorage.setItem("lastname", "Hatton");

You will need to validate that web storage is supported. You would do that by adding this piece of code to your javascript.
if (typeof(Storage) !== "undefined"){ //code to run}
else{
//what to do}
