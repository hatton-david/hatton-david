JSON objects have two basic methods to deal with JSON-formatted data. Parse and stringify.

JSON Parse

When you recieve data from a web server, it will always come in the form of a string. A common use of JSON is to assist in that data exchange to and from a web server. You can use JSON.parse(), to take that data and transform it into a JavaScript object to be used in your program.

JSON.parse() takes a JSON string and transforms it into a JavaScript object.

JSON Parse example

JSON Stringify

JSON.stringify(), however does the opposite. It takes a JavaScript object and transforms it into a JSON string.
