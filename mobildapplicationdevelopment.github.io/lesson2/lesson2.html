There are different ways to create new objects:

  • Define and create a single object, using an object literal.
  • Define and create a single object, with the keyword new.
  • Define an object constructor, and then create objects of the constructed type.

Creating a JavaScript Object.

Inheritance

All JavaScript objects inherit properties and methods from a prototype object.

Objects can inherit all of the properties and behaviors of the parent object. First you need to create the prototype object, then you can make new instances of that object. You can change it further by adding new properties, and then have new objects created from it. This allows you to reuse the code without having to write it over and over again.


function Person(firstName, lastName) {
this.FirstName = firstName || "unknown";
this.LastName = lastName || "unknown";
};

Person.prototype.getFullName = function () {
return this.FirstName + " " + this.LastName;
}


In the example above, I defined Person class function with FirstName and LastName as its properties. I also also added getFullName method to its prototype object.

I will create another class that inherits from the Person class so that we don't have to redefine FirstName, LastName and getFullName() method in this class.


function David(firstName, lastName, hairColor, eyeColor)
{
Person.call(this, firstName, lastName);

this.hairColor = hairColor || "unknown";
this.eyeColor = eyeColor || "unknown";

} David.prototype = new Person();
David.prototype.constructor = David;
