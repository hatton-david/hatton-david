CSS Transitions

CSS transitions give you the ability to change property values smoothly, over a specified duration of time.

In my example, you can hover over the elements below, to see the height change at different speeds.

linear

ease

ease-in

ease-out

ease-in-out

















The transition-timing-function property tells you not only the speed of the transition effect, but can have varying speeds throughout the effect.

Here are some of the values the transition-timing-function property can have

  • ease - Starts slow, then fast, then ends slowly (this is default)
  • linear - Same speed from start to end
  • ease-in - Slow start
  • ease-out - Slow end
  • ease-in-out - Slow start and end
